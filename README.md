# Jewellery Expense Tracker

## Overview

Jewellery Expense Tracker is a simple and efficient web-based application designed to help jewellery businesses record, manage, and track daily expenses. The application provides an easy-to-use interface for maintaining expense records, monitoring spending, and improving financial management.

## Features

* Add new expense records
* Edit existing expenses
* Delete expense entries
* Category-wise expense management
* Automatic total expense calculation
* Search and filter expenses
* Responsive and user-friendly interface
* Local data storage support
* Mobile and desktop compatible

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/jewellery-expense-tracker.git
```

2. Open the project folder.

3. Open `index.html` in your browser.

## Usage

1. Enter expense details.
2. Select the appropriate category.
3. Save the expense record.
4. View and manage expense history.
5. Track total expenses and generate reports.
